Notion 同步配置说明

FlowSpace 的 Notion 同步默认只把带有指定同步标签的 FlowSpace 笔记推送到 Notion；从 Notion 拉回 FlowSpace 需要在同步面板里手动触发。

数据库链接或 Data Source ID 是什么

这里可以直接粘贴 Notion 数据库右上角菜单里的 Copy link 链接，也可以填写 Notion 的 Data Source ID。FlowSpace 会自动从链接里提取数据库 ID，并在需要时解析到真正的数据源。

配置时，先在 Notion 中准备一个用于存放 FlowSpace 笔记的数据库，并把它分享给 Notion integration。你截图里的菜单如果看不到 Data Source ID 入口，就用 Copy link。

不要复制普通 Notion 页面链接。FlowSpace 需要的是数据库本体的链接；如果页面里嵌入了表格或 linked database，请打开那个数据库本体后再复制链接。

如果仍然出现 404，通常不是 ID 格式问题，而是这个数据库还没有在 Connections 里授权给对应 integration。

Notion Token（原始令牌）是什么

这个输入框填写 Notion integration 的原始 Token，通常以 ntn_ 或 secret_ 开头。

保存后 FlowSpace 不会在设置界面回显 Token。再次编辑时留空会继续使用已保存的 Token，输入新 Token 后会覆盖旧值。旧版通过环境变量保存的配置仍可继续使用。

推荐配置步骤

  1. 在 Notion 创建或选择一个用于 FlowSpace 笔记同步的数据库。
  2. 创建 Notion integration，并在数据库的 Connections 里确认已经添加这个 integration。
  3. 复制 integration 的原始 Token。
  4. 在数据库右上角菜单点击 Copy link，把链接粘贴到 FlowSpace 的“数据库链接或 Data Source ID”。
  5. 在 FlowSpace 同步设置中填写 Notion Token。
  6. 如果只想同步部分笔记，在同步标签里填写标签名，并给对应笔记打标签。

如果连接测试失败，优先检查三件事：数据库链接是否来自正确数据库、 integration 是否已在 Connections 里获得访问权限、Notion Token 是否来自正确的 integration。
