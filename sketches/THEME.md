## 一、设计原则

| 原则 | 含义 | 实现方式 |
|------|------|---------|
| **一眼看全** | 仪表盘首屏覆盖任务+日历+笔记 | 三栏网格，无需滚动 |
| **沉浸写作** | 编辑器是内容优先，UI 不抢戏 | 窄栏宽 740px，悬浮工具条 |
| **呼吸感** | 不拥挤，不空旷 | Notion 式留白 + 卡片间距 |
| **悬浮浮现** | 常用操作不用时隐藏 | Hover 显示工具条、添加按钮 |
| **轻量操作** | 完成任务/创建事件 ≤ 2 步 | Checkbox 直接勾 + 底部快捷栏 |

## 四、间距系统

以 4px 为基准网格：

| 令牌 | 值 | 用途 |
|------|-----|------|
| `space-1` | 4px | 紧凑元素间距（标签之间） |
| `space-2` | 8px | 相关元素间距（图标-文字） |
| `space-3` | 12px | 列表项间距 |
| `space-4` | 16px | 卡片内边距、组件间距 |
| `space-5` | 20px | 卡片之间 |
| `space-6` | 24px | Section 之间 |
| `space-8` | 32px | 页面边距 |
| `space-10` | 40px | 大区块间隔 |

圆角：
| 令牌 | 值 | 用途 |
|------|-----|------|
| `rounded-sm` | 3px | 标签、小徽章 |
| `rounded` | 5px | 按钮、输入框 |
| `rounded-md` | 6px | 任务行、check box |
| `rounded-lg` | 8px | 卡片、弹窗 |

## 七、交互模式

### 7.1 悬浮浮现 (Hover Reveal)

编辑器工具条默认 `opacity: 0`，hover 当前 block 时变为 `opacity: 1`：

```css
.editor-toolbar { opacity: 0; transition: opacity 0.15s; }
.editor-block:hover .editor-toolbar { opacity: 1; }
```

同样适用于：
- 笔记列表的「删除/更多」按钮
- 任务行的「编辑」按钮
- 侧边栏文件夹的「…」菜单

### 7.2 Quick Capture 触发

| 方式 | 触发键 |
|------|--------|
| 全局快捷键 | `⌘/Ctrl + Shift + N` |
| 顶部栏按钮 | 点击「⚡ 快速捕获」 |
| 底部快捷栏 | 点击蓝色胶囊按钮 |

### 7.3 键盘快捷方式

| 快捷键 | 动作 |
|--------|------|
| `⌘⇧N` | 打开快速捕获 |
| `⌘K` | 全局搜索 / 命令面板 |
| `⌘⇧[` | 收起/展开侧边栏 |
| `⌘1` | 切换到仪表盘 |
| `⌘2` | 切换到笔记 |
| `⌘3` | 切换到任务 |
| `⌘4` | 切换到日历 |
| `⌘S` | 保存当前笔记 |
| `Escape` | 关闭弹窗 / 退出编辑 |

### 7.4 过渡动画

| 元素 | 动画 | 时长 |
|------|------|------|
| 侧边栏展开/收起 | width 过渡 | 200ms ease |
| 卡片 hover | 边框色 + 阴影 | 150ms ease |
| 快速捕获面板 | slideDown (translateY + opacity) | 200ms ease-out |
| 编辑器工具条 | opacity | 150ms |
| Checkbox 完成 | 背景色 + 缩放 | 200ms ease |

### 7.5 拖拽

| 场景 | 实现方式 |
|------|---------|
| 日历事件调整时间 | 拖拽事件边缘或整块移动 |
| 任务排序 | 拖拽任务行上下重排 |
| 侧边栏文件夹排序 | 拖拽文件夹 |

> v1 不做拖拽排序，v1.1 补上。

## 八、页面状态

### 8.1 空状态

每个列表/视图需处理空状态：

| 视图 | 空状态文案 | 引导操作 |
|------|-----------|---------|
| 仪表盘（无任务） | "今天还没有任务 👀" | 「+ 添加任务」按钮 |
| 笔记列表 | "还没有笔记，开始写第一篇吧" | 「+ 新建笔记」按钮 |
| 任务列表 | "收件箱是空的 🎉" | 「+ 创建任务」按钮 |
| 日历（当天无事件） | "今天没有安排" | 「+ 添加事件」按钮 |
| 搜索结果为空 | "没有找到 'xxx'" | 搜索建议 |

### 8.2 加载状态

| 场景 | 展示方式 |
|------|---------|
| 应用启动 | 骨架屏（仪表盘三栏灰色占位块） |
| 笔记列表加载 | 3 个笔记卡片骨架 |
| 笔记内容加载 | 编辑器区域骨架文本 |
| 搜索中 | 搜索框右侧 spinner |

### 8.3 错误状态

| 场景 | 处理 |
|------|------|
| 文件写入失败 | Toast 通知：「保存失败，请检查磁盘空间」 |
| 数据库异常 | 全屏错误页 + 「重启应用」按钮 |
| 搜索索引损坏 | 静默重建索引，底部状态栏显示进度 |
